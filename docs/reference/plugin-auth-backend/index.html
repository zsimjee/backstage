Interfaces

InterfaceDescription
AuthProviderRouteHandlersAny Auth provider needs to implement this interface which handles the routes in the auth backend. Any auth API requests from the frontend reaches these methods.The routes in the auth backend API are tied to these methods like below/auth/[provider]/start -> start /auth/[provider]/handler/frame -> frameHandler /auth/[provider]/refresh -> refresh /auth/[provider]/logout -> logout
OAuthHandlersAny OAuth provider needs to implement this interface which has provider specific handlers for different methods to perform authentication, get access tokens, refresh tokens and perform sign out.
